# 本番環境用設定一覧

| 設定 | 取得場所 | 登録場所 | 秘密情報 |
|---|---|---|---|
| Streamlit公開URL | Community CloudのApp URL | Secrets `app.public_url`、Supabase Site/Redirect URL | いいえ |
| 許可メール | 先生が使用するGoogleアカウント | Secrets `app.allowed_emails`、Supabase `allowed_teachers` | 個人情報 |
| Supabase Project URL | Project Settings → API | Secrets `supabase.url` | いいえ |
| Supabase Publishable/anon key | Project Settings → API | Secrets `supabase.anon_key` | 公開前提だがGitへ直書きしない |
| Supabase service-role key | Project Settings → API | 移行時だけ環境変数 | はい・最重要 |
| Supabase Callback URL | Authentication → Providers → Google | Google OAuth Authorized redirect URIs | いいえ |
| Google Client ID | Google Auth Platform → Clients | Supabase Google Provider | 準秘密 |
| Google Client secret | Google Auth Platform → Clients | Supabase Google Provider | はい |
| Calendar ID | Google Calendar設定 | Secrets `google_calendar.calendar_id` | 場合により個人情報 |

本番アプリへ登録するのはPublishable/anon keyまでです。service-role keyとGoogle Client secretをGit、アプリコード、READMEへ記載しません。
